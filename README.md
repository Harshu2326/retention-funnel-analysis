# 📊 User Retention & Funnel Analysis

## 📌 Overview

This project analyzes user behavior using funnel analysis, cohort retention, and event distribution.

## 🎯 Objectives

* Understand user drop-off across funnel stages
* Measure retention trends
* Analyze user engagement

## 📊 Key Insights

* Significant drop-off before upgrade stage
* Retention declines after Day 3
* Most activity occurs in login and core features

## 🚀 Business Impact

* Improve onboarding flow
* Increase early engagement
* Optimize upgrade conversion

## 🛠️ Tools Used

* Python (Pandas, Seaborn, Matplotlib, Plotly)
* SQL

## 📁 Project Structure

* `data/` → datasets
* `notebooks/` → analysis
* `sql/` → queries
* `dashboards/` → outputs

## 📌 Conclusion

Improving early-stage engagement can significantly increase retention and conversions.
